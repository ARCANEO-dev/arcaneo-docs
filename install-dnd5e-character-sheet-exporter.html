ARCANEO Installation Guide

How to Install ARCANEO – DnD5e Character Sheet Exporter

You can install the module directly from the Foundry VTT module browser. Two quick installation methods are available, and both are explained below.

Installation Methods

  • Method 1: Install the module directly from the Foundry module browser.
  • Method 2: Install the module using the Manifest URL.

Method 1 — Foundry Module Browser

Step 1 — Open Foundry VTT, go to Add-on Modules, and click Install Module

Step 2 — Search for the module

Search for ARCANEO – DnD5e Character Sheet Exporter.

Step 3 — Click Install

After clicking Install, Foundry will download and install the module.

Once the installation is complete, you will see a confirmation message at the top of the screen, and the module will appear in the installed modules list.

Method 2 — Manifest URL

If the module does not appear yet in your Foundry client, you can install it using the Manifest URL.

Step 1 — Open Foundry VTT, go to Add-on Modules, and click Install Module

Step 2 — Paste the Manifest URL

Paste this URL into the Manifest URL field:

https://raw.githubusercontent.com/ARCANEO-dev/arcaneo-dnd5e-character-sheet-exporter/main/module.json

Step 3 — Click Install

After clicking Install, Foundry will download and install the module.

Once the installation is complete, you will see a confirmation message at the top of the screen, and the module will appear in the installed modules list.

Enable the Module

Once installed, enable the module inside your world.

How to Use It

Open the character sheet you want to export, then go to Toggle Controls.

Then click ARCANEO Export Character Sheet.

Result

Your export is ready.

Compatibility

Compatible with Foundry VTT v13 and the DnD5e system (5.2+).
